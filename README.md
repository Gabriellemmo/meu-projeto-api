# 📚 Primeira API - ASP.NET Core

## Clonar o projeto

```bash
git clone https://github.com/Gabriellemmo/meu-projeto-api.git
cd meu-projeto-api
```

## Executar

```bash
dotnet restore
dotnet run
```

Acesse:

```
https://localhost:5254/swagger
```
